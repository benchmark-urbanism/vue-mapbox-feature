vue-mapbox-feature


A Vue module for dynamic geoJSON feature collections in Mapbox GL JS

Add GeoJSON feature collections to your Vue components: 


    <vue-mapbox-feature
            :map='map'
            :uid='"circle-example"'
            :layer-type='"circle"'
            :feature='pointGeom'
            :paint='circlePaint'
            :pulse='true'
    ></vue-mapbox-feature>

Use custom styling based on the mapbox style spec: 


    circlePaint: {
        'circle-radius': 5,
        'circle-color': '#29434e',
        'circle-opacity': 0.5,
        'circle-stroke-width': 2,
        'circle-stroke-color': '#29434e',
        'circle-stroke-opacity': 0.8
    }

Combine with turf.js for manipulating features: 


    polyGeom () {
        let c = turf.concave(this.pointGeom)
        let b = turf.buffer(c, -this.buffer)
        return b
    }
